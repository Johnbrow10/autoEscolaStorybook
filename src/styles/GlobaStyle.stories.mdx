# Global Style

CSS Reset and base stylesheets definition.

```jsx
const app = () => (
  <>
    <GlobalStyle />
    <OtherComponents />
  </>
);
```

## Typography

Basic global typography

### Headings

All HTML headings, `h1` through `h6`, are available:

<Preview>
  <Story name="Headings">
    <div>
      <h1>Título 1</h1>
      <h2>Título 2</h2>
      <h3>Título 3</h3>
      <h4>Título 4</h4>
      <h5>Título 5</h5>
      <h6>Título 6</h6>
    </div>
  </Story>
</Preview>
